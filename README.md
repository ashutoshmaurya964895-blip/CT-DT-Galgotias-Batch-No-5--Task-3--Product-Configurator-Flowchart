# CT&DT Galgotias - Batch No: 5
## Task#03 – Product Configurator Tool – Flowchart

## 📋 Project Objective
This repository documents our analysis of digital product customizers within the Concept-Knowledge (C-K) Theory framework for Task 3. The flowchart maps the dynamic decision-making paths, sequential feature dependencies, and configuration steps a consumer experiences when building a customized product online.

## ⚙️ Analyzed Configurator Path
Our interactive flowchart visualizes the user choice tree for an industrial product customization matrix, mapping out:
* **Base Concept Entry:** Selecting baseline product models and foundational architectures.
* **Component Branching:** Navigating feature restrictions (e.g., specific options requiring upgraded trim levels).
* **Final Synthesis Gate:** Compiling selections into a unique bill of materials, pricing total, and production-ready order state.
